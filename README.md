# 🎬 Netflix Clone

A modern and responsive Netflix Clone built using HTML5, CSS3, and JavaScript. This project recreates the look and feel of Netflix's landing page while incorporating interactive features, animations, and responsive design principles.

## 🚀 Features

* Responsive Design for Desktop and Mobile
* Netflix-Inspired User Interface
* Interactive FAQ Accordion
* Email Validation Functionality
* Popup Welcome Message
* Smooth Hover Animations
* Trending Movies Section
* Modern Layout and Styling
* Clean and Organized Code Structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📸 Project Preview

This project mimics the Netflix landing page with:

* Hero Banner Section
* Navigation Bar
* Trending Movies Cards
* Reasons to Join Section
* Frequently Asked Questions
* Footer Section
